# Day 1: Basic User Info

This project takes user input and displays formatted personal information.

## Features
- Input name, age, and height
- Calculate age after 5 years
- Convert height from meters to centimeters
- Check if user is Adult or Minor
- Display all results neatly using a list

## Concepts Practiced
- Input and output
- Basic calculations
- Conditional statements
- Lists for organized output
