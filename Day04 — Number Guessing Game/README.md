# Day 4: Number Guessing Game

A simple number guessing game where the user guesses a secret number between 1 and 10.

## Features
- Random number generation
- User input for guesses
- Hints: too low / too high
- Count number of attempts
- Game ends when correct or max attempts reached

## Concepts Practiced
- Functions (`def`)
- While True loops
- Conditional statements
- Counters

```bash
python main.py
